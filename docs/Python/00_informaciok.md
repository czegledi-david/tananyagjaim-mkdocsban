## Python

### Telepítési útmutató

1. Töltse le a legújabb verziót a hivatalos oldalról: **[Python letöltése](https://www.python.org/downloads/)**
2. A letöltött telepítőt futtassuk, ha szükséges, akkor rendszergazdaként.
3. A telepítő elindításakor a kezdőképernyőn be kell pipálni az **'Add python.exe to PATH'** opciót!

!!! warning "Nagyon fontos lépés!"
    Ha a PATH opciót kihagyja a telepítésnél, a számítógép nem fogja felismerni a Python parancsokat, és a fejlesztőkörnyezet nem fog működni!

## Visual Studio Code

### Telepítési útmutató

1. Töltse le a legújabb verziót a hivatalos oldalról: **[Visual Studio Code letöltése](https://code.visualstudio.com/Download)**
2. A letöltött telepítőt futtassuk, ha szükséges, akkor rendszergazdaként.
3. A telepítést követően az órák során szükségünk lesz még kiegészítőkre. A VS Code "Extensions" menüjében (vagy az alábbi linkeken) telepítse ezeket:
    * Python extension: [Letöltés](https://marketplace.visualstudio.com/items?itemName=ms-python.python)
    * Jupyter extension: [Letöltés](https://marketplace.visualstudio.com/items?itemName=ms-toolsai.jupyter)
4. Kezdő lépések: [Python in Visual Studio Code](https://code.visualstudio.com/docs/python/python-tutorial)
