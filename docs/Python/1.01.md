# 2. Szoftverkörnyezet kialakítása

## 2.1. Mire van szükségünk a programozáshoz?

Ahhoz, hogy el tudjunk kezdeni Pythonban programozni, a számítógépünket is fel kell készíteni erre a feladatra. A gépünk ugyanis önmagában, "gyárilag" nem érti a Python utasításait. Két alapvető dologra lesz szükségünk:

1.  **Egy értelmezőre (Interpreter):** Ez a Python "motorja". Ez az a háttérprogram, amely elolvassa az általunk írt emberi kódot, és lefordítja azt a számítógép processzora számára érthető gépi kódra.
2.  **Egy fejlesztőkörnyezetre (IDE / Kódszerkesztő):** Ez az a felület (program), ahol magát a kódot írjuk. Bár elméletileg egy egyszerű Jegyzettömbben is írhatnánk programot, egy dedikált szerkesztő rengeteget segít: kiszínezi a parancsokat, és azonnal jelzi a legordítóbb hibákat.

## 2.2. A Python letöltése és telepítése

A hivatalos Python értelmezőt teljesen ingyenesen le lehet tölteni.

1.  Látogass el a hivatalos weboldalra: [python.org/downloads](https://www.python.org/downloads/)
2.  Töltsd le az operációs rendszerednek (pl. Windows) megfelelő legújabb verziót!
3.  Indítsd el a telepítőt.
4.  **Kritikus lépés:** A legelső telepítési ablak alján mindenképpen pipáld be az **"Add Python to PATH"** (vagy hasonló nevű) jelölőnégyzetet! Ez biztosítja, hogy a gépeden minden program megtalálja majd a Pythont.
5.  Kattints az *Install Now* gombra, és várd meg, amíg a folyamat befejeződik.

## 2.3. A fejlesztőkörnyezet kiválasztása: Az IDLE

A Python telepítésekor automatikusan és ingyenesen megkapjuk a nyelv saját, beépített fejlesztőkörnyezetét, az **IDLE**-t (Integrated Development and Learning Environment). 

A tanulásunk során, illetve **az órákon és a feladatok megoldásához mi végig ezt fogjuk használni.**
Az IDLE kifejezetten kezdőknek és oktatási célokra készült: rendkívül letisztult, nem vonja el a figyelmedet ezerféle felesleges funkcióval és gombbal, ugyanakkor stabilan tud mindent, amire a programjaink megírásához és teszteléséhez szükségünk van.

*(Érdekességképpen: A profi fejlesztők a nagy szoftvercégeknél gyakran használnak hatalmas, robusztus programokat a kódíráshoz, mint amilyen például a **PyCharm**. Ez egy ipari szabványnak számító, professzionális eszköz nagyon összetett projektekhez, azonban a programozás alapjainak elsajátításakor a túlzott bonyolultsága miatt inkább csak akadályozná a tanulást.)*

### 💻 Gyakorlati feladat

Most, hogy tudjuk az elméletet, piszkoljuk be a kezünket! A mai óra célja, hogy a géped készen álljon a kódolásra, és futtasd életed első programját az IDLE segítségével.

1.  **Telepítés:** Telepítsd fel a Pythont a gépedre a leírásban szereplő lépések alapján (ha még nincs telepítve az iskolai gépen)! Ne felejtsd el bepipálni az "Add to PATH" opciót!
2.  **Az IDLE elindítása:** Keresd meg a Windows Start menüjében a *Python* mappát, és kattints az **IDLE (Python 3.x 64-bit)** ikonra! Ekkor megnyílik egy fehér ablak, a *Python Shell*. Ez a tesztelő felületünk, ami azonnal reagál a parancsokra.
3.  **A valódi kód megírása:** A komolyabb programokat nem a Shellbe írjuk, hanem külön fájlba mentjük. Az IDLE felső menüsorában kattints a `File -> New File` opcióra! Ekkor megnyílik egy üres, tiszta szerkesztőablak.
4.  **Mentés és programozás:** Mentsd el ezt az üres fájlt a számítógépeden (pl. az Asztalra vagy egy új mappába) `File -> Save As...` menüponttal, mondjuk `elso.py` néven!
    *   Most írd be a következő egyetlen sort pontosan így a szerkesztőablakba (figyelve az idézőjelekre és a kisbetűkre): 
        `print("Hello, ez az első Python programom!")`
5.  **Futtatás:** Nyomd meg az **F5** billentyűt a billentyűzeten (vagy válaszd a felső menüből a `Run -> Run Module` opciót)! A program lefut, és a végeredményt az eredeti (Shell) ablakban fogod látni. Gratulálok, hivatalosan is működő szoftverkörnyezeted van!
